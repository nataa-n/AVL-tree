# AVL-tree

A generic, self-balancing binary search tree (AVL tree) written in C#. It supports insert, search, and delete, all running in O(log n), and stays balanced through rotations even when values are inserted in sorted order. Because it is generic, it works with any comparable type such as int or string. The project includes xUnit tests covering search, deletion, sorted traversal, generic use, and the O(log n) height. Run them with dotnet test.
